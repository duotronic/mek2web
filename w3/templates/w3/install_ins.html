Instructions to download & install QkJob

Tested with Microsoft Windows Desktop 7, 8 and 10 (QkJob is easily removed.)

QkJob Install from web - Windows 7

·       From http://www.ret-tek.com, click on “Installer”

·       The Microsoft windows install dialog appears – Ret-Tek / QkJob

·       Click on the “Install” button

·       In the dialog at the bottom of the screen, click “Run”

·       In the dialog “Application Install – Security Warning”

·       Click “Install”

·       QkJob starts downloading (< minute?)

·       Once QkJob is installed, it will start automatically;

·       (Please close the installer window)

QkJob Install from web - Windows 8

·          From http://www.ret-tek.com, click on “Installer”

·          A Microsoft windows install dialog appears – Ret-Tek / QkJob

·          Click on the “Install” button

·          In the dialog at the bottom of the screen, click “Run”

·          In the screen banner “Windows protected your pc”

·          Click “More info”

·          Click “Run anyway”

·          In the “Application Install – Security Warning

·          Click “Install”

·          The “installing” dialog starts (a couple minutes)

·          In the screen banner “Windows protected your pc”

·          Click “More Info”

·          Click “Run Anyway”

·          Once QkJob is installed, it will start automatically;

·          (Please close the installer window)

·          To invoke QkJob, look for Ret-Tek in the applications list, QkJob is in there!

Thank you!
